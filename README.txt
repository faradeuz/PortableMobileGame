===================README==========================
Основной целью является адаптировать веб-приложение,
которое требуется разработать с помощью технологий
HTML5, Javascript, css. К приложению предъявляется 
минимальные требования по функциональности. 

Предпологаемые этапы разработки проекта:
1- разработка прототипа кроссплатформенного мобильного 
   приложения
2- составления product backlog, sprint backlog, 
   scrum table
3- составление и реализация 5 unit-тестов, используя 
   библиотеку Jasmine
4- используя фреймворк Adobe PhoneGap, преобразовать 
   разработанное приложение в Android-приложение.
5- тестирование файла-сборки .apk на эмуляторе Android
